# Hanamori Spirit · 花守灵契

- First-party creator: **Animacraft Atelier**
- Canvas: `1024 x 1024`
- Visual direction: contemporary Japanese cel-shaded spirit-garden portrait
- License intent: free personal Soul mint and avatar use with Maker provenance retained; the final published Sui policy governs commercial use and resale.

## Part inventory

- Background: 4 Items

Total public Items: **25**. The pack supports **5,120** valid default-rule combinations.

## Production notes

- The source atlases use a flat green key and are never served to players.
- Runtime layer PNGs are generated at one common origin with alpha transparency.
- Hair tufts are stylized hair silhouettes, not literal animal anatomy.
- Clothing is original fantasy fashion and is not presented as a real religious uniform.
- `animacraft-manifest.json` includes editable Soul Character, Memory, and Skills defaults for the Soulidity handoff.

## AI disclosure

OpenAI's built-in image generation path created the original master and layer atlases. Animacraft Atelier supplied prompt design, art direction, selection, chroma cleanup, atlas slicing, composition correction, background design, metadata, and QA. No third-party Maker image was provided as a model input.
