# Hanamori Spirit prompt set

## Master

Create one polished original Japanese cel-shaded anime bust for a layered OC Maker: a gender-neutral shrine-garden spirit keeper with jade eyes, a chestnut long bob with fox-ear-like tufts made only from hair, vermilion-and-cream fantasy robes, and a plum blossom ornament. Use clean line art, two-step cel shadows, refined textile detail, a centered front-facing modular pose, and a flat `#00ff00` chroma background. Avoid named artists, existing characters, literal animal ears, real-uniform claims, logos, text, painterly rendering, photorealism, and 3D.
