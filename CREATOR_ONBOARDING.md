# Animacraft Creator Onboarding

## Goal

Ship one usable OC template that lets users create many distinct characters from the same artist base.

## Required Template Info

- Template name
- Creator display name
- Style or world
- License type
- Royalty BPS
- Short summary
- Cover preview

## Asset Pack Rules

- Format: transparent PNG.
- Canvas: 1024 x 1024 px.
- Anchor: centered at 512,512 unless documented otherwise.
- Naming: `slot_part_variant.png`.
- Recommended slots: `background`, `hairBack`, `hairFront`, `eyes`, `mouth`, `outfit`, `accessory`.
- Every layer must align on the same canvas without manual repositioning.

## Authorization Options

- `personal-use`: users can make avatars and personal OC images.
- `free-remix`: users can remix and share derived OC outputs.
- `paid-commercial`: users can pay for commercial usage under creator terms.
- `exclusive-commission`: one buyer or project receives limited exclusive usage.

## Revenue Fields

- `royaltyBps`: creator share in basis points. Example: `300` means 3%.
- `price`: creator-facing template pricing note.
- `license`: authorization rule users must see before export.

## Launch Checklist

- Metadata is complete.
- At least 20 PNG parts are ready for a first public template.
- Each major slot has at least 3 choices.
- File names follow `slot_part_variant.png`.
- No layer has a solid background.
- Works preview includes at least 4 example OCs.
- License and revenue terms are visible in template detail.
